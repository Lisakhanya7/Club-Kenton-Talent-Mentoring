## Using the Admin Dashboard

### 1. **Fixtures & Results Management**

#### Adding a Fixture (Upcoming Match)
1. Click on **"Fixtures & Results"** in the sidebar
2. Fill in the form:
   - **Match Type**: Select the competition (League Division 1, U18 Championship, etc.)
   - **Date & Time**: When the match is scheduled
   - **Opponent Team**: The team KTM will play against
   - **Venue**: Where the match will be played
   - **Age Group**: (Optional) U12, U14, U16, U18
   - **Home/Away**: Is it a home or away game?
3. Click **"Add Fixture"**
4. The fixture will automatically appear on the main website

#### Adding a Match Result
1. Fill in the results form:
   - **Competition**: Which tournament/league
   - **Date**: When the match was played
   - **Opponent**: The opposing team
   - **Goals**: Enter KTM goals and opponent goals
   - **Goalscorers/Notes**: (Optional) Names of who scored
2. Click **"Add Result"**
3. Latest results will show on the website in the correct section

### 2. **Squad Management**

#### Adding a Player
1. Click on **"Squad Management"** in the sidebar
2. Fill in the player details:
   - **Name**: Player full name
   - **Position**: Goalkeeper, Defender, Midfielder, Forward, or Striker
   - **Age**: Player's current age
   - **Jersey Number**: Their playing number (1-99)
   - **Goals, Assists, Appearances**: Season statistics
3. Click **"Add Player"**
4. Players automatically appear on the Squad page, sorted by jersey number

### 3. **News & Updates**

#### Publishing an Article
1. Click on **"News & Updates"** in the sidebar
2. Create your article:
   - **Category**: Match Report, Training Update, Club Announcement, Achievement, Transfer News, or Academic Success
   - **Date**: Publication date
   - **Title**: Article headline
   - **Content**: Full article text
3. Click **"Publish Article"**
4. Articles appear on the website sorted by newest first

### 4. **Media Gallery**

#### Adding Media Items
1. Click on **"Media Gallery"** in the sidebar
2. Add a media item:
   - **Type**: Video Highlight, Match Photo, or Interview
   - **Date**: When it was published
   - **Title**: Name of the video/photo set/interview
   - **Duration/Details**: (Optional) e.g., "2:45 min" or "12 photos"
3. Click **"Add to Gallery"**
4. Media appears organized by type on the main website

2. **Update Results Quickly**: Add match results the same day as the game

## Troubleshooting

### Data Not Appearing?
- Clear browser cache and refresh the page
- Check if you're using the same browser
- Make sure localStorage is enabled in browser settings

### Lost Data?
- Check browser's local storage (only available on that device/browser)
- If deleted, you'll need to re-add it

### Forms Not Submitting?
- Make sure all required fields (marked with *) are filled
- Check browser console for errors (F12 > Console tab)
